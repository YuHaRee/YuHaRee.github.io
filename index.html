<CampUs> 프로젝트

소개: 팀 단위로 캠핑장 예약 시스템을 개발하여 사용자가 간편하게 캠핑장을 조회하고 예약할 수 있도록 구현한 프로젝트입니다.

  • 기간: 2024.07 ~ 2024.08 (6주)
  • 인원: 4명
  • 기술 스택:
    - 언어: Java 21, JavaScript, Python
    - 데이터베이스: MySQL 8.0, DynamoDB
    - 프레임워크: Spring Boot 3.1, Spring Security
    - 서버: AWS EC2, AWS Lambda, AWS ELB
    - 프론트엔드: Vue.js, HTML5, CSS3
    - 테스트: Locust, Elastic APM, Swagger UI
  • 주요 기능:
    - 캠핑장 정보 조회 및 필터링 기능
    - Redis를 이용한 예약 관리 및 동시성 제어
    - Spring Security와 AWS Cognito를 통한 사용자와 관리자 인증
    - API 문서화를 위해 Swagger UI 사용
    - Locust를 이용한 성능 테스트 및 Elastic APM을 통한 모니터링
  • GitHub: Camp_us 프로젝트

<날씨 알림> 프로젝트

소개: 사용자가 설정한 지역에 대한 실시간 날씨 정보를 제공하고, 날씨 변화에 따른 알림을 발송하는 시스템입니다.

  • 기간: 2024.06 ~ 2024.07 (3주)
  • 인원: 4명
  • 기술 스택:
    - 언어: Java 17, JavaScript, Python
    - 데이터베이스: MySQL
    - 프레임워크: Spring Boot 3.1
    - 서버: AWS Lambda, AWS EC2, SMTP
    - 프론트엔드: React, HTML5, CSS3
  • 주요 기능:
    - OpenWeather API를 사용한 실시간 날씨 데이터 조회
    - AWS Lambda와 SMTP를 활용한 이메일 알림 발송
    - 사용자 설정에 따른 지역별 날씨 필터링
    - 트래픽 분산 및 확장성을 위한 AWS EC2와 ELB 구성
  • GitHub: Weather 알림 시스템

<꽃 판매업> 프로젝트

소개: 사용자에게 꽃을 구매하고 배달할 수 있는 자유로운 플랫폼을 제공하는 프로젝트입니다.

  • 기간: 2022.02 ~ 2022.04 (8주)
  • 인원: 4명
  • 기술 스택:
    - 언어: Java 8, JavaScript
    - 데이터베이스: MySQL
    - 프레임워크: Spring Boot 2.6.4
    - 서버: Apache Tomcat 9.0
    - 프론트엔드: HTML5, CSS3, AJAX
  • 주요 기능:
    - 사용자 회원가입 및 로그인 구현
    - 꽃 등록 및 수정, 삭제 기능 제공
    - 장바구니에 꽃 추가 및 결제 기능
    - 주문 내역 및 배송 상태 확인
